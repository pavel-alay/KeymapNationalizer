# [KeymapNationalizer](https://plugins.jetbrains.com/plugin/14625-keymap-nationalizer)

Custom version with additional predefined rules for [Universal Layout](https://github.com/tonsky/Universal-Layout/):
```
; with $
' with ^
\ with "
/ with &
` with '
1 with !
3 with #
4 with ;
6 with :
8 with *
9 with (
0 with )
```
